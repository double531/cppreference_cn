#C++ reference 中文版

##About

[cppreference.com](http://en.cppreference.com/w/)所谓的中文版乃是 google translate的结果，你懂得，于是禀着现学现用现翻译的原则，用到哪部分就先翻译哪部分，不保证进度和顺序，不分先后。

然而，并不是纯粹的翻译，亦可能会加入自己的理解，所以可能会有不对的地方，欢迎纠错！

> 带链接的为已翻译部分！
